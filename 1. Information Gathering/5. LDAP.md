### ldapsearch
```bash
# Print all infos
ldapsearch -x -H ldap://<IP_ADDRESS> -D '' -w '' -b "DC=<SUBDOMAIN>,DC=<TLD>"
# Extract Users
ldapsearch -v -x -b "DC=<SUBDOMAIN>,DC=<TLD>" -H "ldap://<IP_ADDRESS>" "(objectclass*)" | grep sAMAccountName:
# Print all infos with authentication
ldapsearch -x -H ldap://<IP_ADDRESS> -D '<DOMAIN>\<USERNAME>' -w '<PASSWORD>' -b "DC=<SUBDOMAIN>,DC=<TLD>"
```

### netexec, nxc
```bash
# Enumerate users
netexec ldap <IP_ADDRESS> -u '' -p '' --users
# Enumerate users with output file
netexec ldap 10.10.10.172 -u '' -p '' --users-export output.txt
# Enumerate just actived users
netexec ldap 10.10.10.172 -u '' -p '' --active-users
```
